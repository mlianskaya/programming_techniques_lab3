## Описание
В работе реализованы три модифицированных генератора псевдослучайных чисел:

- **MLCG‑XOR** – линейный конгруэнтный генератор с модулем 2⁶⁴ и нелинейным перемешиванием (XOR + сдвиги).  
- **LFG‑XOR** – генератор Фибоначчи с запаздыванием (лаги 55 и 24) с операцией XOR и улучшенной инициализацией.  
- **Xorshift128‑MLT** – Xorshift128 с дополнительным нелинейным умножением (техника SplitMix64).

Для каждого генератора сформировано 20 выборок по 20 000 целых чисел. Для каждой выборки вычислены среднее, стандартное отклонение, коэффициент вариации и выполнена проверка равномерности распределения по критерию хи‑квадрат (уровень значимости 0.05). Все выборки объединены в одну битовую строку, к которой применены пять тестов NIST (частотный, блочный частотный, на серии, на самую длинную серию, кумулятивных сумм). Проведены замеры времени генерации от 1 000 до 1 000 000 чисел для каждого генератора.

### Результаты
- Все три генератора показали равномерное распределение (хи‑квадрат пройден для 95–100% выборок).
- Тесты NIST выявили дефекты: MLCG‑XOR и LFG‑XOR не прошли тесты на самую длинную серию и кумулятивных сумм; Xorshift128‑MLT не прошёл только тест кумулятивных сумм.
- По скорости: MLCG‑XOR самый быстрый (≈0,94 мс на 10⁶ чисел), Xorshift128‑MLT ≈2,78 мс, LFG‑XOR ≈3,01 мс.
