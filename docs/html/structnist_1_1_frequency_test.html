Detailed Description

Частотный (monobit) тест — проверяет, близка ли доля единиц во всей последовательности к 1/2.

Статистика: S_n = (количество единиц) - (количество нулей). Нормированная статистика: S = |S_n| / √n. p-value = erfc(S/√2). Если p-value ≥ α, последовательность случайна.
