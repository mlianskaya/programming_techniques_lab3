Detailed Description

Генератор Фибоначчи с запаздыванием и XOR (LFG-XOR).

Базовый алгоритм: LFG с лагами (55,24). Модификации:

  • Вместо сложения используется XOR (быстрее, лучше перемешивание).
  • Добавлена нелинейная функция scramble после обновлений.
  • Улучшенная инициализация буфера через LCG и перемешивание.

Инициализация буфера с заданным seed.

Инициализация буфера генератора.

Parameters
seedначальное значение для инициализации.

Алгоритм:

  1. Заполняем буфер размера LAG1 (55) с помощью LCG (state = state*1103515245+12345). Каждое значение сразу пропускается через scramble для начального перемешивания.
  2. Затем 5 раз проходим по буферу: для каждого i берём элемент с отставанием j = (i + LAG2) % LAG1, делаем XOR и снова scramble. Это гарантирует, что все начальные значения хорошо перемешаны и нет зависимости от seed.

Returns

Если индекс достиг конца буфера (index >= LAG1), то:

  • проходим по всему буферу, для каждого i обновляем buffer[i] по правилу buffer[i] ^= buffer[(i + LAG2) % LAG1], затем применяем scramble.
