Detailed Description

Тест кумулятивных сумм — проверяет, насколько сильно отклоняется накопленная сумма (присваивая биту '1' +1, биту '0' –1) от нуля.

Вычисляются максимальное положительное и отрицательное отклонения S_max и S_min. Нормированные значения: z1 = S_max / √n, z2 = |S_min| / √n. p-value = P(z) = Σ_{k=-∞}^{∞} [erfc((2k+1)z/√2) - erfc((2k-1)z/√2)]. Окончательный p-value = min(P(z1), P(z2)). Малое p-value означает, что накопленная сумма слишком часто (или слишком редко) пересекает нуль.
