Вычисляет статистику хи-квадрат и число степеней свободы для выборки.

Template Parameters
NumberЧисловой тип элементов выборки (должен поддерживать сравнение, преобразование к double).
Parameters
sampleВектор наблюдаемых значений.
minМинимальное возможное значение (включительно).
maxМаксимальное возможное значение (включительно для UNIFORM_INT, исключительно для UNIFORM_REAL).
typeТип распределения (целочисленное или непрерывное).
Returns
std::pair<double, std::size_t> Первый элемент — значение хи-квадрат, второй — число степеней свободы.
Exceptions
std::invalid_argumentесли min >= max, или размер выборки < 2, или значение выборки вне диапазона.
