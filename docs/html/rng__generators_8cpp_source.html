54// 2. LFG‑XOR: генератор Фибоначчи с запаздыванием и XOR
55// Принцип: LFG использует рекуррентную формулу
56// X[n] = X[n-p] XOR X[n-q] (p=55, q=24). Такой выбор лагов даёт
57// большой период (~2^79) и хорошую случайность.
58// Модификации:
59// - Вместо сложения используется XOR, что быстрее и даёт лучшее перемешивание битов.
60// - Добавлена функция scramble (XOR + сдвиги) для нелинейного преобразования
61// каждого нового значения, что улучшает статистические свойства.
62// - Инициализация буфера выполняется с помощью LCG, а затем делается несколько
63// проходов перемешивания, чтобы сделать генератор менее чувствительным к seed.
