\doxysubsection{Detailed Description}
Модифицированный LCG с нелинейным перемешиванием (MLCG-\/\+XOR). 

Классический LCG\+: X\+\_\+\{n+1\} = (a \texorpdfstring{$\ast$}{*} X\+\_\+n + c) mod 2\texorpdfstring{$^\wedge$}{\string^}64. Модификации\+:
\begin{DoxyItemize}
\item Модуль 2\texorpdfstring{$^\wedge$}{\string^}64 за счёт естественного переполнения uint64\+\_\+t.
\item Параметры a и c подобраны для полного периода 2\texorpdfstring{$^\wedge$}{\string^}64.
\item После каждого шага применяется нелинейное перемешивание (XOR и сдвиги), устраняющее корреляцию младших битов. 
\end{DoxyItemize}
