\doxysubsection{Detailed Description}
Тест на самую длинную серию — проверяет, не является ли максимальная длина серии в блоках слишком большой (что указывает на неслучайность). 

Последовательность разбивается на блоки по 8 бит. Для каждого блока находится максимальная длина серии (подряд идущих одинаковых битов). Затем сравнивается наблюдаемое распределение максимальных длин с теоретическим. Статистика χ² рассчитывается по 7 категориям (длина ≤1, 2, 3, 4, 5, 6, ≥7). p-\/value вычисляется приближённо как exp(-\/χ²/2).
