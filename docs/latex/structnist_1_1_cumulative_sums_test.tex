\doxysubsection{Detailed Description}
Тест кумулятивных сумм — проверяет, насколько сильно отклоняется накопленная сумма (присваивая биту \textquotesingle{}1\textquotesingle{} +1, биту \textquotesingle{}0\textquotesingle{} –1) от нуля. 

Вычисляются максимальное положительное и отрицательное отклонения S\+\_\+max и S\+\_\+min. Нормированные значения\+: z1 = S\+\_\+max / √n, z2 = \texorpdfstring{$\vert$}{|}\+S\+\_\+min\texorpdfstring{$\vert$}{|} / √n. p-\/value = P(z) = Σ\+\_\+\{k=-\/∞\}\texorpdfstring{$^\wedge$}{\string^}\{∞\} \mbox{[}erfc((2k+1)z/√2) -\/ erfc((2k-\/1)z/√2)\mbox{]}. Окончательный p-\/value = min(P(z1), P(z2)). Малое p-\/value означает, что накопленная сумма слишком часто (или слишком редко) пересекает нуль.
