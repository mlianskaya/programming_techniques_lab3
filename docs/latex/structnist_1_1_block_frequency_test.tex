\doxysubsection{Detailed Description}
Блочный частотный тест — проверяет равномерность распределения единиц внутри блоков фиксированного размера. 

Последовательность разбивается на N непересекающихся блоков размера M. Для каждого блока вычисляется доля единиц π\+\_\+i. Статистика\+: χ² = 4M Σ (π\+\_\+i -\/ 0.\+5)². Приближённо распределена как χ² с N степенями свободы. Для больших N используется нормальная аппроксимация.
