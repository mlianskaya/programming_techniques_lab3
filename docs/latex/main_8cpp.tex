\doxysubsection{Detailed Description}
Программа для тестирования трёх модифицированных генераторов псевдослучайных чисел. 


\begin{DoxyEnumerate}
\item Генерирует 20 выборок по 20 000 элементов для каждого генератора.
\item Для каждой выборки вычисляет среднее, стандартное отклонение и коэффициент вариации.
\item Проверяет равномерность распределения каждой выборки критерием хи-\/квадрат (уровень значимости 0.\+05).
\item Применяет пять тестов NIST (частотный, блочный частотный, на серии, на длину серии, кумулятивных сумм).
\item Замеряет время генерации от 1 000 до 1 000 000 чисел и сохраняет результаты в CSV.
\item Результаты хи-\/квадрат и NIST выводятся в консоль. 
\end{DoxyEnumerate}
