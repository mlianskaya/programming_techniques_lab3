\doxysubsection{Detailed Description}
Реализация пяти статистических тестов из набора NIST SP 800-\/22. 

Набор тестов предназначен для проверки случайности битовых последовательностей. Реализованы следующие тесты\+:
\begin{DoxyItemize}
\item Frequency (Monobit) Test (проверка глобальной доли единиц)
\item Block Frequency Test (проверка равномерности внутри блоков)
\item Runs Test (проверка количества серий)
\item Longest Run Test (проверка максимальной длины серии)
\item Cumulative Sums Test (проверка случайности накопленной суммы)
\end{DoxyItemize}

Все тесты вычисляют p-\/value. Если p-\/value ≥ заданного уровня значимости, последовательность считается случайной.
